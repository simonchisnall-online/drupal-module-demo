INTRODUCTION
------------

The Kat Autocloud modules connects to the car yard api and pulls down stock to be created as nodes on this drupal site.


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install the Chaos Tool Suite module as you would normally install a
   Drupal module. Visit
   https://www.drupal.org/docs/extending-drupal/installing-modules
   for further information.


CONFIGURATION
-------------

No configuration is needed but you can go to /admin/config/autocloud if you want to test API or manually run an update or delete stock.
